    3. ロールバック機能の詳細検証¶

    現在提供されているロールバック方法¶

    方法1: PowerShellコマンド（ドキュメント記載）¶

    問題点¶
    • ❌ 自動化されていない - 手動コマンド実行が必要
    • ❌ エラー処理なし - 失敗時の処理が不明
    • ❌ 検証なし - 復元後の確認プロセスがない

    改善が必要なロールバック機能¶

    def rollback_from_backup(backup_timestamp: str = None):
    """
    バックアップからの復元
    """
    backup_base = Path("temp/backups")

    if backup_timestamp:
        backup_dir = backup_base / backup_timestamp
    else:
        # 最新のバックアップを探す
        backups = sorted(backup_base.glob("*"), reverse=True)
        if not backups:
            return False, ["バックアップが見つかりません"]
        backup_dir = backups[0]

    # 復元処理
    errors = []
    for backup_file in backup_dir.glob("*"):
        try:
            target = Path(backup_file.name)
            shutil.copy2(backup_file, target)
        except Exception as e:
            errors.append(f"復元失敗: {backup_file.name} - {e}")

    return len(errors) == 0, errors
