    スクリプト一覧と使用方法¶

    最終更新: 2025/09/16

    1. 本番使用スクリプト（correct/ディレクトリ）¶

    1.1 一般・自動・UD・マルチ部門用¶

    module_b_replace_existing_FIXED.py¶

    • 目的: 既存画像参照の置換
    • 状態: ✅ 修正済・使用可
    • 機能: image4-image19の参照を直接パス形式に変換
    • 重要修正: 削除ロジックの修正（行全体でなく参照のみ削除）

    module_c_add_new_images_FIXED.py¶

    • 目的: 新規画像の追加
    • 状態: ✅ 修正済・使用可
    • 機能: 指定位置に6個の新規画像を挿入
    • 重要修正: 重複挿入の防止、正確なマーカー位置特定

    module_d_remove_definitions.py¶

    • 目的: base64画像定義の削除
    • 状態: ✅ 使用可
    • 機能: ファイル末尾の[imageN]:定義を完全削除

    1.2 ビギナー部門用¶

    module_beginner_fix_FINAL.py¶

    • 目的: ビギナー部門の画像参照修正
    • 状態: ✅ 最終版・使用可
    • 機能:
    • image1-image8の参照を変換
    • 共通設備画像を正しく認識
    • base64定義の削除
    • 重要点: 相対パス対応、共通設備画像の考慮

    2. 調査・分析スクリプト（investigation/ディレクトリ）¶

    analyze_image_references.py¶

    • 目的: 画像参照パターンの分析
    • 使用時期: 初期調査

    extract_base64_images.py¶

    • 目的: base64画像の抽出と確認
    • 使用時期: 画像内容の検証時

    3. 廃止スクリプト（deprecated/ディレクトリ）¶

    ⚠️ 以下は使用禁止

    • module_beginner_fix.py （誤ったマッピング）
    • module_beginner_fix_CORRECT.py （パス問題あり）
    • replace_images_correctly.py （初期版、問題あり）

    4. 実行順序¶

    一般・自動・UD・マルチ部門¶

    cd docs/temp/correct
python module_b_replace_existing_FIXED.py
python module_c_add_new_images_FIXED.py
python module_d_remove_definitions.py

    ビギナー部門¶

    cd docs/temp/correct
python module_beginner_fix_FINAL.py

    5. 共通仕様¶

    5.1 全スクリプト共通¶

    • UTF-8エンコーディング対応
    • 自動バックアップ作成
    • 作業ファイル（_WORK.md）での安全な操作
    • 検証機能付き

    5.2 画像パス形式¶

    • 旧: ![][imageN]
    • 新: ![](images/category/filename.png)
    • キャプションなし（ファイル名が説明的）

    6. トラブルシューティング¶

    エラー: ファイルが見つからない¶

    • 相対パスを確認（correct/から実行時は../../が必要）

    エラー: エンコーディング¶

    • UTF-8を明示的に指定

    エラー: 画像が表示されない¶

    • MkDocsの画像パスは docs/ からの相対パス
    • ファイル名の日本語文字を確認

    7. 検証コマンド¶

    # 旧形式の確認（0であるべき）
grep -c "!\[\]\[image" ファイル名.md

# 新形式の確認
grep -c "!\[\](images/" ファイル名.md

# base64の確認（0であるべき）
grep -c "data:image/png;base64" ファイル名.md

    注: スクリプト実行前に必ずバックアップを確認すること
